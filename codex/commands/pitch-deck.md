# /pitch-deck

Сборка презентации для инвестора/партнёра/грантодателя.

## Intake

1. Для кого? (инвестор / ФСИ / Сколково / АСКОН / партнёр)
2. Повод? (знакомство / промежуточный отчёт / демо-день / заявка)
3. Длительность? (5 / 10 / 20 минут)
4. Акцент? (технология / команда / рынок / финансы)

## Параллельные задачи (strategy ‖ accounting)

1. **strategy-agent** — позиционирование и структура:
   - Проблема → решение → дифференциатор
   - Целевой рынок, конкурентные преимущества
   - Достижения и планы
   - Один ключевой «слайд-хук»

2. **accounting-agent** — финансовые слайды:
   - Burn rate, остаток средств
   - Unit-экономика (если есть)
   - Финансирование: грант ФСИ, Сколково
   - Потребность в инвестициях / следующий раунд

ПАРАЛЛЕЛЬНО.

## Последовательно (контент)

3. **pitch-deck** skill — сборка структуры под конкретную аудиторию (разная под инвестор vs ФСИ).

## Evaluator

Для внешних аудиторий (инвестор, ФСИ) — обязательный ревью. Особенно проверяем: цифры из accounting правдивы, нет рекламных преувеличений.

## Файл

4. **Codex presentations** — собрать .pptx файл с брендингом.

## Формат

- `.pptx` в `documents/presentations/pitch_[аудитория]_YYYY-MM-DD.pptx`
- Черновик слайдов: `documents/presentations/pitch_[аудитория]_YYYY-MM-DD.md`

## Верификация

- Все цифры имеют источник (не «из головы»)
- Нет маркетинговых преувеличений про команду/продукт
- Соответствие формату аудитории (инвестор vs ФСИ очень разный)
- Evaluator: PASS
