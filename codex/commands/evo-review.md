# /evo-review

Ручной еженедельный анализ самообучения системы (Learning Cycle).

**Режим**: ручной. Запускается руководителем каждую пятницу. Через 4 недели — переход на scheduled task (по решению руководителя).

## Процесс

Запускает `python scripts/learning_cycle.py --mode weekly` и показывает отчёт.

Скрипт выполняет:

1. **Агрегация calibration_log за последние 7 дней**:
   - Категории ошибок (9 типов)
   - Какой субагент ошибся чаще
   - Повторяющиеся паттерны (Critical Failure Step)

2. **Анализ тенденций**:
   - Новые категории ошибок
   - Регрессы по сравнению с предыдущей неделей
   - Успешные улучшения из system_evolution (работают ≥ 5 задач?)

3. **Предложение правок (L1/L2)**:
   - L1 — правка промпта субагента
   - L2 — новый чек-лист / golden-test
   - L3 — структурное (требует явного подтверждения)
   - Для каждого: subagent, file, reason, expected_effect, risk

4. **Cross-Domain check**:
   - Применимо ли найденное правило к другим субагентам?

5. **Snapshot подготовка**:
   - Перед применением любой L1 — создаётся backup в `codex/roles/backups/[agent]_YYYY-MM-DD_[reason].md`
   - НО правка НЕ применяется без явного ОК от руководителя (первые 4 недели)

6. **Feedback-правила в Compound Knowledge Base**:
   - Каждую ошибку конвертировать в feedback-entity (через memory_cli)
   - Пометить применимость (subagent, task_type)

## После запуска скрипта

Диспетчер:
1. Показывает руководителю отчёт (markdown)
2. Для каждого предложения правки — спрашивает: применить / отложить / отклонить
3. При «применить» — диспетчер делает snapshot + Edit + запись в system_evolution.md
4. При отклонении — записывает в «Отклонённые предложения» system_evolution

## Формат результата

- `operations/evo_review_YYYY-MM-DD.md` — полный отчёт
- Обновление `operations/system_evolution.md` — если применили
- Обновление `memory/entities/feedback.jsonl` — новые правила

## Верификация после применения

- Прогнать golden-tests затронутого агента → сравнить с baseline (regression?)
- Если regression > 10% → автооткат из snapshot, запись в «Откаченные изменения»
