# grant-tracker

Трекинг грантовых обязательств и финансов ФСИ «Старт-ЦТ-1».

## Когда использовать

- "Что по гранту?" — общий статус
- Проверка расходов план/факт
- Проверка выполнения вех
- Ближайшие дедлайны по гранту
- Подготовка данных для отчёта (для fsi-report)

## Intake

1. Что нужно? (общий статус / расходы / вехи / дедлайны / конкретный вопрос)
2. За какой период? (если про расходы)

## Шаги

1. Загрузить из памяти: entities типа obligation, связанные с ФСИ
2. Загрузить из памяти: entities типа fact, связанные с грантом
3. Проверить documents/grants/ — план разработки, смета
4. Проверить DATA/ — финансовые данные (если есть)
5. Сформировать запрошенный отчёт

## Форматы результата

### Общий статус
```
Грант ФСИ «Старт-ЦТ-1» — статус на [дата]

Сумма: [X] ₽
Освоено: [Y] ₽ ([Z]%)
Текущий этап: [N] из [M]
Ближайшая веха: [название] — [дата]
Ближайший отчёт: [тип] — [дата]
Открытые риски: [кол-во]
```

## Связь с другими скиллами

- `fsi-report` — использует данные grant-tracker для формирования отчёта
- `accounting-agent` — источник финансовых данных

## Запись в память

Записывать: изменения статуса вех, новые обязательства по гранту.
НЕ записывать: расчёты (они в файлах), промежуточные данные.
