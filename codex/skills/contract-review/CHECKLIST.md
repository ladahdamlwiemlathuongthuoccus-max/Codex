# Чек-лист проверки договора

Загружается при self-check анализа договора.

## Обязательные пункты для проверки

### Стороны и реквизиты
- [ ] Стороны указаны корректно (наименование, ИНН, ОГРН)?
- [ ] Реквизиты совпадают с ЕГРЮЛ?
- [ ] Подписант имеет полномочия?

### Предмет договора
- [ ] Предмет однозначно определён?
- [ ] Результат работ/услуг описан конкретно?
- [ ] Критерии приёмки указаны?

### Сроки
- [ ] Срок действия договора?
- [ ] Сроки выполнения этапов?
- [ ] Порядок продления / расторжения?

### Финансовые условия
- [ ] Общая сумма?
- [ ] График платежей?
- [ ] Условия оплаты (аванс, постоплата)?
- [ ] Валюта, НДС?

### Ответственность
- [ ] Неустойка за просрочку?
- [ ] Ограничение ответственности?
- [ ] Форс-мажор?

### Интеллектуальная собственность
- [ ] Кому принадлежат результаты?
- [ ] Право на модификацию?
- [ ] Лицензионные условия?

### Конфиденциальность
- [ ] NDA включён или отдельный?
- [ ] Срок конфиденциальности?
- [ ] Что считается конфиденциальным?

### Прочее
- [ ] Порядок разрешения споров?
- [ ] Применимое право?
- [ ] Порядок внесения изменений?
