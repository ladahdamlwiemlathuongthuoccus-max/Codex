# Чек-лист комплектности отчёта ФСИ

Загружается при проверке комплектности документов.

## Документы отчёта

- [ ] Научно-технический отчёт (описание выполненных работ)
- [ ] Финансовый отчёт (расходы по статьям сметы)
- [ ] Акт о выполнении этапа
- [ ] Аннотированный отчёт

## Подтверждающие документы (финансовые)

- [ ] Платёжные поручения по каждому расходу
- [ ] Договоры с подрядчиками / поставщиками
- [ ] Акты выполненных работ / оказанных услуг
- [ ] Счета-фактуры (при наличии НДС)
- [ ] Расчётные ведомости по зарплате (если есть статья ФОТ)
- [ ] Трудовые договоры / ГПХ (для привлечённых специалистов)

## Подтверждающие документы (технические)

- [ ] Результаты испытаний / тестирования
- [ ] Скриншоты / демонстрация работоспособности
- [ ] Исходный код (если требуется)
- [ ] Документация к ПО

## Соответствие смете

- [ ] Каждый расход соответствует утверждённой статье сметы
- [ ] Нет превышения по статьям > допустимого %
- [ ] Общая сумма расходов ≤ суммы финансирования за этап
- [ ] Софинансирование подтверждено (если требуется)

## Формальные требования

- [ ] Отчёт подписан руководителем проекта
- [ ] Нумерация страниц
- [ ] Ссылки на договор и номер заявки корректны [ПРОВЕРИТЬ НОРМУ — уточнить актуальные требования ФСИ через веб-поиск]
