# Path Policy

## Разрешённая рабочая зона

Codex-система может свободно создавать, изменять и тестировать файлы только внутри:

`D:\REDPEAK\Agent systems\Agent for Codex\Agent Codex`

## За пределами рабочей зоны

Любая запись, удаление, перемещение, правка конфигов или установка, меняющая файлы вне рабочей зоны, требует отдельного явного разрешения пользователя.

Особо защищённые пути:

- `D:\REDPEAK\Agent systems\AgentHQ`
- `C:\Users\sashatrash\.codex\config.toml`
- `C:\Users\sashatrash\.codex\plugins`
- `C:\Users\sashatrash\.claude`

## Правило для глобального Codex config

Если нужна интеграция в `C:\Users\sashatrash\.codex\config.toml`:

1. Сформировать точный фрагмент TOML внутри `codex/mcp/`.
2. Показать пользователю, что будет добавлено или изменено.
3. Запросить отдельное разрешение.
4. Только после подтверждения применять изменение.
