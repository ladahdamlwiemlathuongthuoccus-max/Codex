# Ежемесячный чек-лист бухгалтера

## До 25-го числа каждого месяца

- [ ] Рассчитать взносы за директора: МРОТ (27 093) x 30% = 8 128 руб
- [ ] Рассчитать взносы за исполнителей ГПХ (если были выплаты)
- [ ] Рассчитать НДФЛ за исполнителей ГПХ (если были выплаты)
- [ ] Подготовить уведомление по страховым взносам для отправки через СБИС
- [ ] Подготовить уведомление по НДФЛ (если были выплаты)

## До 28-го числа каждого месяца

- [ ] Оплатить страховые взносы на ЕНС
- [ ] Оплатить НДФЛ на ЕНС (если были выплаты)

## Ежеквартально (до 25-27 числа месяца после квартала)

- [ ] Декларация по НДС (нулевая / освобождение Сколково)
- [ ] Декларация по налогу на прибыль (0% Сколково)
- [ ] РСВ (расчёт по страховым взносам)
- [ ] 6-НДФЛ
- [ ] ЕФС-1 раздел 2 (взносы на травматизм)
- [ ] Расчёт суммы налоговых льгот резидента Сколково (новая форма с 2026)

## При выплате по ГПХ

1. Рассчитать gross: сумма_на_руки / (1 - 0.13)
2. Удержать НДФЛ: gross x 13%
3. Рассчитать взносы:
   - Часть до 1,5 МРОТ (40 640): x 30%
   - Часть свыше 1,5 МРОТ: x 15% (льгота Сколково)
4. Перечислить: сумму на руки исполнителю, НДФЛ на ЕНС, взносы на ЕНС
5. Оформить: акт выполненных работ, платёжное поручение

## При получении гранта ФСИ

- [ ] Открыть отдельный аналитический счёт для учёта целевого финансирования
- [ ] Вести реестр расходов по гранту (дата, сумма, статья, исполнитель, документ)
- [ ] Контролировать лимиты: ИП/самозанятые до 25% от гранта, ГПХ до 150 000 руб/мес на человека
- [ ] Сохранять все первичные документы (договоры, акты, платёжки)

## Годовые действия (начало следующего года)

- [ ] Бухгалтерский баланс + отчёт о финансовых результатах (до 31 марта)
- [ ] Годовая декларация по прибыли (до 25 марта)
- [ ] Годовой 6-НДФЛ (до 25 февраля)
- [ ] ЕФС-1 сведения о стаже (до 25 января)
- [ ] Подготовить отчёт по гранту ФСИ (по срокам договора с Фондом)
