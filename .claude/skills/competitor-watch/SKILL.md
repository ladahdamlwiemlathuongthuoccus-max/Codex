# competitor-watch

Конкурентный анализ и мониторинг рынка CAD/rule-based/ML для инженерии.

## Когда использовать

- Анализ конкретного конкурента
- Обзор конкурентного ландшафта
- Подготовка battle card для переговоров
- Сравнение подходов (наш vs конкуренты)
- Для презентации инвесторам (раздел "рынок")

## Intake

1. Что нужно? (анализ конкурента / обзор рынка / battle card / сравнение)
2. Конкретные компании или продукты? (или искать)
3. Фокус: технология / бизнес-модель / позиционирование / всё?
4. Для кого результат? (для себя / инвестор / партнёр)

## Шаги

1. Загрузить из памяти: предыдущие конкурентные анализы (если есть)
2. Веб-поиск: актуальная информация о конкурентах
3. Сформировать анализ по шаблону
4. Self-check: факты проверены через веб-поиск, а не из общих знаний
