# pitch-deck

Подготовка контента и структуры презентаций.

## Когда использовать

- Презентация для инвестора / фонда
- Презентация для партнёра (АСКОН и др.)
- Демо-день / конкурс
- Отчётная презентация для ФСИ / Сколково
- Внутренняя презентация для команды

## Intake

1. Для кого? (инвестор / партнёр / грантодатель / команда / конкурс)
2. Цель презентации? (привлечь деньги / партнёрство / отчитаться / информировать)
3. Сколько слайдов / минут?
4. Есть ли конкретные требования к формату?
5. Нужен .pptx файл или только структура с текстом?

## Автоопределение структуры

### Для инвестора (10-12 слайдов)
1. Проблема
2. Решение
3. Как это работает (демо/скриншоты)
4. Рынок (размер, динамика)
5. Бизнес-модель
6. Трекшн / результаты
7. Конкуренты
8. Команда
9. Roadmap
10. Финансы (запрос / unit-экономика)
11. Контакты

### Для партнёра (6-8 слайдов)
1. Кто мы
2. Что делаем
3. Зачем партнёрство (win-win)
4. Как выглядит интеграция
5. Текущие результаты
6. Предложение / следующие шаги

### Для ФСИ / Сколково (по требованиям)
1. Загрузить из памяти: обязательства и требования к отчётности
2. Следовать формату, указанному в документах гранта

## Правила

- Одна мысль на слайд
- Заголовок слайда = главный тезис (не тема, а вывод)
- Цифры крупно, текст минимально
- Без длинных абзацев на слайдах
- Каждый слайд отвечает на вопрос "и что?"

## Шаги

1. Определить аудиторию и цель
2. Загрузить из памяти: релевантные факты, метрики, решения
3. Сформировать структуру по шаблону
4. Написать контент для каждого слайда (заголовок + тезисы + заметки спикера)
5. Self-check: каждый слайд имеет конкретный тезис, а не абстракцию
6. Если нужен .pptx — передать в pptx-скилл

## Запись в память

НЕ записывать содержимое презентации. Записывать: ключевые метрики, которые использовались (для актуализации в будущем).
