# Золотые тест-кейсы: accounting-agent

Используются при обновлении промпта субагента для проверки, что качество не ухудшилось.

---

## Тест 1: Расчёт взносов

**Вход:** «Рассчитай страховые взносы и НДФЛ с выплаты 100 000 ₽ по договору ГПХ.»

**Ожидаемый результат:**
- Конкретные суммы: НДФЛ, ОПС, ОМС, на руки
- Ставки взяты из скилла accounting-agent/RATES.md (с указанием источника)
- Промежуточные шаги расчёта показаны
- Учтён статус Сколково (если применим, с указанием основания)

**Красные флаги:**
- Неверные ставки (не из RATES.md)
- Отсутствие промежуточных вычислений
- Итог не сходится с суммой по позициям
- Выдуманные ставки или льготы

---

## Тест 2: Сверка расходов с бюджетом гранта

**Вход:** «Покажи план/факт расходов по гранту ФСИ за Q1 2026.»

**Ожидаемый результат:**
- Таблица: статья бюджета, план (из сметы), факт (из documents/finance/), отклонение
- Файл сметы указан конкретно
- Каждый расход привязан к первичному документу
- Итоговые суммы арифметически верны

**Красные флаги:**
- Статьи бюджета, которых нет в смете
- Выдуманные суммы расходов
- Отсутствие ссылки на файл сметы
- Арифметические ошибки в итогах

---

## Тест 3: Комплектность документов

**Вход:** «Проверь комплектность первички за февраль 2026.»

**Ожидаемый результат:**
- Чек-лист: документ, есть/нет, где лежит
- Если документа нет — [ДАННЫЕ ОТСУТСТВУЮТ] с указанием, что нужно и куда положить

**Красные флаги:**
- Утверждения о наличии документа без проверки файла
- Пропуск обязательных документов
