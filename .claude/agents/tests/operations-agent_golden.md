# Золотые тест-кейсы: operations-agent

Используются при обновлении промпта субагента для проверки, что качество не ухудшилось.

---

## Тест 1: Недельный план

**Вход:** «Составь план на неделю 31 марта — 4 апреля. Формат — тезисы.»

**Ожидаемый результат:**
- 3-7 конкретных задач с датами/дедлайнами
- Каждая задача привязана к источнику (файл или запись в памяти)
- Учтены актуальные обязательства из памяти
- Указаны ближайшие внешние дедлайны (ФСИ, Сколково, контрагенты)

**Красные флаги:**
- Задачи без источников («нужно подготовить отчёт» — какой? откуда взял?)
- Выдуманные дедлайны
- Отсутствие проверки operations/risk_log.md
- Абстрактные формулировки вместо конкретных действий

---

## Тест 2: Подготовка к созвону

**Вход:** «Подготовь к созвону с CAD-разработчиком. Формат — повестка + открытые вопросы.»

**Ожидаемый результат:**
- Загружены: контакт разработчика из памяти, обязательства по договору, текущие статусы задач
- Повестка из 3-5 пунктов с контекстом по каждому
- Открытые вопросы привязаны к конкретным обязательствам/вехам

**Красные флаги:**
- Нет ссылки на файл договора
- Общие вопросы без привязки к реальным задачам («как дела?», «есть ли проблемы?»)
- Пропущены актуальные обязательства из памяти

---

## Тест 3: Дедлайны

**Вход:** «Покажи все дедлайны на ближайшие 2 недели.»

**Ожидаемый результат:**
- Таблица: дата, обязательство, источник, статус
- Данные из памяти (obligations) + documents/grants/
- Отсортировано по дате

**Красные флаги:**
- Дедлайны без указания источника
- Выдуманные даты
- Пропущены обязательства, которые есть в памяти
